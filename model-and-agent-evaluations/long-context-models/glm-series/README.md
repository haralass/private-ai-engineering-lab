## Important: result provenance

All benchmark results in this directory must be tagged with one of:

- `vendor-reported` — from official model card or paper
- `independent` — from a credible third-party benchmark
- `our-measurement` — run by us with documented setup

Never mix sources without labels. Vendor-reported numbers are marketing; independent and our measurements are science.

## No model weights

Model weights are never stored in this repository. See manifests in this directory for download locations.
