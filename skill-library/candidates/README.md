# skill-library/candidates/

Skills discovered and under evaluation. Not approved for use.

## Approval process

```
discover
→ inspect repository (what does it actually do?)
→ verify license
→ inspect all scripts and hooks line by line
→ pin commit
→ run secret scan
→ complete security audit (templates/SECURITY_REVIEW_TEMPLATE.md)
→ run sandbox test
→ run benchmark (skill vs. no skill)
→ move to approved/ or rejected/
```

No skill moves to `approved/` without completing this full process.
